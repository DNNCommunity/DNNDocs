FacebookRequest

Gets or sets facebookRequest property is filled when request is coming though Facebook iFrame (e.g. fan pages).

Declaration
FacebookRequest FacebookRequest { get; set; }
Property Value
Type Description
FacebookRequest
Remarks

FacebookRequest property is populated based on data in "signed_request" headers coming from Facebook. In order to ensure request is coming from Facebook, FacebookRequest.IsValidSignature method should be called with the secret key provided by Facebook. Most of the properties in IClientCapability do not apply to Facebook.
