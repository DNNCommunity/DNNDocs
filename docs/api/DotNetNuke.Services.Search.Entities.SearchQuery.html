WildCardSearch

Gets or sets a value indicating whether set this to true to perform perform WildCard Search.

Declaration
public bool WildCardSearch { get; set; }
Property Value
Type Description
System.Boolean
Remarks

This property is not respected when Keywords contain special boolean phrases "~", "", "&quot;", "&apos;", "and", "or", "+", "-". When this is enabled, an additional OR is performed, e.g. (keyword OR keyword). It adds asterisk at then end to find any words starting with the keyword. There can be performance implications with this setting turned on.
