Class DnnUnsortedList

Creates a control that renders its childs as a bulleted list.

Remarks

Control renders an unordered list HTML contol. Each child control in DnnUnsortedList is rendered as a separate list item. To obtain a control over list item style, add a DnnUnsortedListItem to a controls list, and tune this object appropriately.
