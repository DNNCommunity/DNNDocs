XPathLiteral(String)

Produce an XPath literal equal to the value if possible; if not, produce an XPath expression that will match the value.

Note that this function will produce very long XPath expressions if a value contains a long run of double quotes.

Declaration
public static string XPathLiteral(string value)
Parameters
Type Name Description
System.String value

The value to match.
Returns
Type Description
System.String

If the value contains only single or double quotes, an XPath literal equal to the value. If it contains both, an XPath expression, using concat(), that evaluates to the value.
