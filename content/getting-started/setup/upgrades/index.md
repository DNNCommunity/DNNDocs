# DNN Platform Upgrades
Following are the basic steps for upgrading **DNN Platform**.

> [!NOTE]
> Every upgrade scenario is unique in some shape, form or fashion. This document is by no means intended to address every scenario. Some upgrades can be quite complex and it is virtually impossible to cover every scenario. This document is intended to focus only on the upgrade basics.

## Upgrade Process by Version

The upgrade process for **DNN Platform** varies depending on which version you are upgrading from:

### For DNN Platform versions prior to 10.2.0

If you are upgrading from a version of DNN Platform **prior to 10.2.0**, please follow the detailed steps outlined in the [Pre-10.2.0 Upgrade Guide](xref:setup-upgrades-pre-10.2.0).

### For DNN Platform versions 10.2.0 and later

If you are upgrading from a version of DNN Platform **10.2.0 or later**, please follow the detailed steps outlined in the [Post-10.2.0 Upgrade Guide](xref:setup-upgrades-post-10.2.0).
