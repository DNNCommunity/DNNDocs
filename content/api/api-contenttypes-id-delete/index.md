# DELETE Content Types with ID

## DELETE /api/ContentTypes/{id}

Deletes the specified content type.

## HTTP Request

In the URL,

```

https://dnnapi.com/content/api/ContentTypes/{id}

```

id

\[string\] The unique identifier of the content type to delete.
